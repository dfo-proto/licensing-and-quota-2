How to try it

  • The licence number is pre-filled with 1006001 (3 results). Select a recipient or two.
  • Clear the search and search 18465805000 for a different set, then select more.
  • Open the Selected recipients tab — your earlier picks are still there. The tab count keeps a running total across searches.
  • Use Clear all (with a confirm) to reset selections. Screen-reader announcements fire on every add, remove and clear.
